## 🏛️ About This Project

**CIVWATCH Community Edition** is the open-source civic intelligence platform that powers transparent, citizen-driven oversight of government and political processes.

This repository contains the community-maintained distribution of CIVWATCH — the same adversarial infrastructure designed to monitor powerful political and financial actors, made available for public contribution and deployment.

> **Full mission and documentation**: [CIVWATCH Main Repository](https://github.com/POWDER-RANGER/CIVWATCH)

---

## 🎯 What It Does

| Module | Function |
|--------|----------|
| **📊 Political Finance Monitor** | Track campaign contributions, PAC activity, and dark money flows |
| **🏛️ Lobbying Tracker** | Analyze LD-2/LD-203 filings to reveal influence networks |
| **🗳️ Voting Record Correlator** | Cross-reference votes with contributions and lobbying contacts |
| **📋 Promise Tracker** | Extract, monitor, and score political promises with evidence |
| **📹 Body Camera Monitor** | Track 18,000+ police BWC policies with compliance scoring |
| **🛡️ Overreach Defense** | Tools to protect citizens from surveillance and abuse |

## 🚀 Quick Start

```bash
# Clone this community edition
git clone https://github.com/POWDER-RANGER/civwatch-powder-ranger.git
cd civwatch-powder-ranger

# Install dependencies
npm install

# Configure
cp .env.example .env
# Edit .env with your API keys

# Run development server
npm run dev
```
